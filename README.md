## 🚀 Featured Portfolio Projects

### 📂 1. Intelligent Document Lifecycle & Process Automation Engine
An asynchronous, webhook-driven multi-agent microservice engineered to automate backend corporate workflows. It transforms unstructured data payloads (messy PDFs, invoices, and scans) into strictly validated, deterministic data structures ready for ERP ingestion.
* **Key Mechanisms:** Non-blocking FastAPI `BackgroundTasks`, mathematical line-item invariant validation via Pydantic, and automated routing to Human-in-the-Loop (HITL) queues if LLM confidence scores drop below 85%.
* 📑 **[View Project Blueprint PDF](./docs/Intelligent_Document_Automation_Engine_Specification.pdf)**
* 📁 *Codebase Location: `/intelligent-document-engine`*

### 📂 2. Enterprise CRM-to-ERP Agentic Hub (Salesforce & Business Central)
An integration gateway acting as an automated autonomous operations layer. It monitors Salesforce case webhooks, triages intents, resolves general billing queries via MS Dynamics 365 Business Central ledger checks, and isolates technical bugs for engineering escalation.
* **Key Mechanisms:** Token-secured endpoints, zero-shot intent classification matching, and deterministic escalation matrix rules guarding against LLM hallucinations in customer support.
* 📑 **[View Project Blueprint PDF](./docs/CRM_to_ERP_Agentic_Hub_Specification.pdf)**
* 📁 *Codebase Location: `/crm-erp-integration-hub`*
